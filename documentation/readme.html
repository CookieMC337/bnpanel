BNPanel is an web application,
and is free software (GNU/GPLv3). It's translated into more 30 languages,

Technically, BNPanel is a web application written in PHP that stores data in a
MySQL database. Users access it using a web browser.

If you would like to know more or help develop this software, please visit
our homepage at http://www.chamilo.org

Portability

BNPanel is an AMP software. This means it should work on any platform running Apache
+ MySQL + PHP. It is then supposed to work on the following Operating Systems :

  • Linux
  • Windows (98, Me, NT4, 2000, XP, VISTA)
  • Unix
  • Mac OS X

It has been tested on

  • Fedora, Mandrake, Red Hat Enterprise Server, Ubuntu, Debian
  • Windows XP, Windows 2000
  • Mac OS X 10.3
We officially recommend Debian and Ubuntu operating systems for their security, community and availability to the public.

E-mail functions remain silent on systems where there is no mail sending software
(Sendmail, Postfix, Hamster...), which is the case by default on a Windows machine.

Interoperability

Client side, BNPanel runs on any browser : Firefox, MS Internet Explorer (5.0+), Netscape (4.7+),
Mozilla (1.2+), Safari, Opera, ...
For better user experience, we recommend Firefox 3.5+ (you can download it freely from http://getfirefox.com).
